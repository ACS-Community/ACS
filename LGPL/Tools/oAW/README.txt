Eclipse setup
=============

Normal Java development installation of Eclipse
  EMF should be included

From Eclipse update side (e.g. Help / Software Updates... / Available Software / Ganymede Update Site), select
  Models and Model Development
    UML2 End-User Features
    UML2 Extender SDK

Install oAW from update site
http://www.openarchitectureware.org/updatesite/milestone/site.xml
